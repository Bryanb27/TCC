# Explorando Tecnologias de Containers no Desenvolvimento de uma Aplicação de Chat

## Descrição
Este repositório contém o código-fonte e a documentação associada ao meu Trabalho de Conclusão de Curso (TCC) sobre computação em nuvem, explorando o uso do Docker e ferramentas da Microsoft Azure. O TCC investiga como as tecnologias de contêineres e a infraestrutura em nuvem podem ser combinadas para fornecer soluções escaláveis e eficientes.

## Tecnologias Utilizadas
- Docker
- Azure Container Registry
- Azure SQL
- Azure Web App
-

## Programas Utilizados
- Visual Studio Code
- Docker Desktop
- Azure Data Studio

## Estrutura do Repositório
1. `codigo/`: Contém o código-fonte do projeto.
2. `documentacao/`: Contém a documentação do projeto, incluindo manuais, relatórios e apresentações.
3. `imagens/`: Contém imagens utilizadas na documentação ou na apresentação.
4. `scripts/`: Contém scripts úteis para implantação, automação ou configuração.
